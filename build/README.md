Build scripts for mod_perimeterx packages.

Currently it produces packages for:
* Debian 8 “Jessie”
* Debian 9 “Stretch”
* CentOS / RHEL 7
* Ubuntu 14.04
* Ubuntu 16.04
* Ubuntu 17.10

To build packages, run:
```shell
./build.sh
```
from `build` directory.

Built packages will be located in `packages/` subfolders.
